What's New in PDFDancer

Templates can now include images — drop in logos, signatures, or photos alongside text, all in a single API call. Plus: image redaction, vector path grouping, and better font handling.


TEMPLATE IMAGE REPLACEMENT
───────────────────────────

You can now replace template placeholders with images, not just text. Drop in logos, signatures, photos — anything. Specify width, height, or both, and mix image and text replacements in a single call.

Use case: Generate branded certificates or invoices where each document gets a personalized name, date, and company logo — all in one API call.

Python:

    pdf.apply_replacements({
      "{NAME}": {"text": "Jane Smith", "font": Font("MyFont", 12)},
      "{LOGO}": {"image": Path("logo.png"), "width": 150},
    })

TypeScript:

    await pdf.replace('{NAME}', { text: 'Jane Smith', font: new Font('MyFont', 12) })
      .andImage('{LOGO}', 'logo.png', 150)
      .apply();

Java:

    pdf.replace("{NAME}", new Font("MyFont", 12), "Jane Smith")
      .replaceWithImage("{LOGO}", new File("logo.png"), 150)
      .apply();

Tip: Use registerFont() to register a full TrueType font before replacements — this avoids missing characters when the PDF's embedded font is subsetted. We've also added CID font support for wider PDF compatibility.


IMAGE FILL REGION
─────────────────

Fill rectangular pixel regions within images with a solid color — without replacing the entire image.

Use case: Black out a face, license plate, or watermark in a scanned document before sharing it externally.

Python:

    images[0].fill_region(10, 10, 50, 30, Color(0, 0, 0))

TypeScript:

    await images[0].fillRegion(10, 10, 50, 30, new Color(0, 0, 0));

Java:

    images.get(0).fillRegion(10, 10, 50, 30, Color.BLACK);


VECTOR PATH GROUPING
────────────────────

Group multiple vector paths together and manipulate them as a single unit. Move, scale, rotate, resize, or remove entire groups in one call. Group by explicit path IDs or by bounding region.

Use case: Company logos in PDFs are often vector paths. Group them and move, scale, or remove the logo as a single object — just like you would with an image.

Python:

    region = {"x": 70, "y": 710, "width": 100, "height": 100}
    group = page.group_paths_in_region(region)
    group.scale(1.5)

TypeScript:

    const region = new BoundingRect(70, 710, 100, 100);
    const group = await page.groupPathsInRegion(region);
    await group.scale(1.5);

Java:

    BoundingRect region = new BoundingRect(70, 710, 100, 100);
    PathGroupReference group = page.groupPathsInRegion(region);
    group.scale(1.5);
